พบช่องโหว่ใหม่ใน OS!!

มีการเปิดเผยช่องโหว่ใหม่ในระบบปฏิบัติการของสมาร์ทโฟนทั้ง iOS และ Android ที่ถูกมิจฉาชีพใช้เพื่อหลอกขโมยข้อมูล Biometric ของผู้ใช้งาน เช่น Face ID และ Fingerprint โดยไม่รู้ตัว ข้อมูลเหล่านี้สามารถนำไปใช้ยืนยันตัวตนในแอปธนาคารจริงได้โดยตรง

กลโกงที่ใช้ Invisible Interface Ad

รายงานพบว่า มิจฉาชีพใช้เทคนิคที่เรียกว่า Invisible Interface Ad ซึ่งเป็นการสร้างหน้าต่างโฆษณาปลอมแบบโปร่งใส (Invisible) หรือปุ่มปิดโฆษณา (Fake Close Button) บนหน้าเว็บไซต์หรือแอป เมื่อผู้ใช้กดปุ่มเหล่านี้ หน้าต่างจะปิดตัวลงเหมือนไม่มีอะไรเกิดขึ้น แต่ในความเป็นจริงแล้ว:

  • แอปพลิเคชันถูกดาวน์โหลดโดยอัตโนมัติ: แอปดังกล่าวจะใช้ช่องโหว่ของระบบปฏิบัติการในการรันไฟล์ติดตั้งแบบอัตโนมัติโดยไม่แจ้งให้ผู้ใช้ทราบ
  • ใช้สิทธิ์ที่ผู้ใช้อาจเคยอนุญาตไว้ใน Cookie หรือการตั้งค่าเว็บไซต์: เช่น “อนุญาตให้ติดตั้งแอปจากแหล่งที่ไม่รู้จัก”
  • ไม่มีการแจ้งเตือน: แอปที่ติดตั้งแล้วจะไม่แสดงไอคอนบนหน้าจอหลัก ทำให้ผู้ใช้ไม่รู้ตัวว่าแอปกำลังทำงานอยู่ในเบื้องหลัง
  • ขโมยข้อมูล Biometric: แอปที่ติดตั้งแล้วจะจำลองการยืนยันตัวตนปลอม เช่น หน้าจอที่ดูเหมือนกำลังใช้ Face ID หรือ Fingerprint ในการปลดล็อก

เมื่อผู้ใช้ทำการปลดล็อก ข้อมูล Biometric จะถูกส่งไปยังเซิร์ฟเวอร์ของมิจฉาชีพ และใช้เพื่อเข้าสู่ระบบแอปธนาคารจริง

ผู้ใช้มักไม่รู้ตัวว่าแอปถูกดาวน์โหลด

ตามที่เคยมีรายงานก่อนหน้านี้ ผู้เสียหายส่วนใหญ่มักระบุว่า "ไม่ได้ดาวน์โหลดแอปใด ๆ" แต่สูญเสียเงินในบัญชีธนาคารอย่างปริศนา ซึ่งในเคสนี้พบว่ากลโกงดังกล่าวทำให้ดูเหมือนว่าไม่มีการดาวน์โหลดใด ๆ เกิดขึ้น โดยใช้ช่องโหว่ของระบบปฏิบัติการร่วมกับ Invisible Interface Ad ในการทำงาน

คำเตือนสำหรับผู้ใช้งาน

เพื่อป้องกันตัวจากการถูกหลอกในลักษณะนี้ ผู้เชี่ยวชาญแนะนำ:

  • อย่ากดปุ่มปิดโฆษณาปลอม: หากพบโฆษณาที่ไม่น่าเชื่อถือ ควรปิดหน้าเว็บหรือแอปพลิเคชันทันที
  • ตรวจสอบสิทธิ์ที่อนุญาตให้แอปพลิเคชัน: หลีกเลี่ยงการเปิดให้ติดตั้งแอปจากแหล่งที่ไม่รู้จัก
  • ใช้ระบบปฏิบัติการรุ่นล่าสุด: การอัปเดตระบบปฏิบัติการช่วยลดช่องโหว่ที่มิจฉาชีพอาจใช้โจมตี
